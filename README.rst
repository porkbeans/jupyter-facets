jupyter-facets
==============

Providing `facets <https://github.com/PAIR-code/facets>`_ wrapper for jupyter


Installation
============
::

   pip install ipyfacets

Usage
=====
Import::

    import ipyfacets as facets

Facets Overview::

    #
    facets.overview({'train': df_train, 'test': df_test})

Facets Dive::

    facets.dive(df)
